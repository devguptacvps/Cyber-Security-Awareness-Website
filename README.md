# Cyber-Security-Awareness-Website
CyberShield is a cybersecurity awareness web app that helps users understand phishing attacks, malware threats, and password security through interactive tools and quizzes.
CyberShield is a web-based cybersecurity awareness application designed to educate users about common online threats and promote safe digital practices. The project focuses on enhancing user understanding of cybersecurity concepts through interactive tools and visually engaging content.

The platform includes multiple modules such as a Phishing URL Detector, Password Strength Checker, Malware Awareness Page, and an Interactive Quiz System. The phishing detector simulates real-world threat analysis using a scanning animation and basic URL evaluation logic. The password checker provides real-time feedback with a dynamic strength bar to encourage strong password creation. The malware section delivers detailed information about different types of cyber threats along with prevention tips.

Additionally, the quiz module helps users test their knowledge with timed questions and provides feedback based on performance, improving engagement and learning outcomes. The entire application is designed with a modern neon-themed user interface to create a realistic cybersecurity environment.

This project is developed using HTML, CSS, and JavaScript, focusing on frontend development, user experience, and practical implementation of cybersecurity awareness concepts.
